## Center a div

```jsx
// Using Flex
<div className="flex items-center justify-center">
  <div>Centered</div>
</div>

// Using Grid
<div className="grid place-items-center">
  <div>Centered</div>
</div>

// Using Position
<div className="relative">
  <div className="absolute -translate-y-1/2 -translate-x-1/2 top-1/2 left-1/2">Centered</div>
</div>
```
