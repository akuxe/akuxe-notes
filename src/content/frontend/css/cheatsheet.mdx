## CSS Selectors

- `h1`: **Tag Selector**
- `#world`: **ID Selector** (highest specificity, unique per page)
- `.heading`: **Class Selector** (used on multiple elements, medium specificity)
- `h1, #world, .heading`: **Multiple Selector** (selects multiple elements)
- `div p`: **Descendant Selector** (nested children within `div`)
- `div > p`: **Child Selector** (direct children of `div`)
- `h1 + h2`: **Adjacent Selector** (selects `h2` immediately after `h1`)
- `p ~ h2`: **General Sibling Selector** (all `h2` after `p` but not nested)
- `input[type="email"]`: **Attribute Selector**
- `*`: **Universal Selector**

## Box Sizing

```css
*,
*::before,
*::after {
  box-sizing: border-box; /* Default: content-box */
}
```

- **Content-box**: Width/height applies only to content.
- **Border-box**: Includes padding and border in width/height.

## Positioning

- `relative`: Relative to the element's normal position.
- `absolute`: Relative to the parent element.
- `fixed`: Relative to the viewport.
- `sticky`: Positioned based on scroll.

```css

  top: 50px;
  left: 50px;
  z-index: 1;
}
```

## Clip Path

- The clip-path property allows you to make complex shapes in CSS by clipping an element to a basic shape (circle, ellipse, polygon, or inset), or to an SVG source.

## Transform

We use to the transform property to modify the element.

```css
section {
  transform: translate(-24px, 18px); /* x, y */
  transform: scale(1.2);
  transform: rotate(45deg);
  transform: skew(10deg, 5deg);
  transform-origin: top right; /* anchor point */

  /* Shorthand has 4 Properties */
  transform: scale(1) skew(0) translate(0, 0) rotate(0);

  /* We can also use matrix */
  transform: matrix(scaleX, skewY, skewX, scaleY, translateX, translateY);
}
```

## Transition

Transitions should be triggered using hover effect, click events, scroll.

```css
section {
  width: 100px;
  transition-property: width;
  transition-duration: 0.2s;
  transition-delay: 0.5s;
  transition-timing-function: ease-in;

  /* Shorthand has 4 Properties */
  transition: property duration easing-function delay;

  /* Multiple Transitions */
  transition:
    width 0.2s ease-in 0.1s,
    height 0.3s ease-out 300ms;
}

section:hover {
  width: 200px;
}
```

## Flexbox

The CSS Flexbox Layout should be used for one-dimensional layout, with rows OR columns.

- [Tailwind CSS](https://tailwindcss.com/docs/flex-basis)
- [W3Schools](https://www.w3schools.com/css/css3_flexbox.asp)

A flex container has two axes:

- Main Axis (default is horizontal, X-axis)
- Cross Axis (default is vertical, Y-axis)

### Flex parent Properties

```css
section {
  display: flex;
  justify-content: space-between; /* Aligns items along the main axis */
  align-items: flex-start; /* Aligns items along the cross axis */
  align-content: center; /* Aligns groups of items on the cross axis */
  flex-flow: row wrap; /* Shorthand for flex-direction: row; and flex-wrap: wrap; */
}
```

### Flex child Properties

```css
article {
  order: 1; /* Changes the order of the item */
  align-self: flex-end; /* Aligns a single item on the cross axis (overrides align-items) */

  flex-grow: 1; /* Item grows to fill available space (default 0, 0 not growable)*/
  flex-shrink: 2; /* Item shrinks if needed (default 1, 0 not shrinkable)*/
  flex-basis: 50%; /* Initial length of the item */

  flex: 1 2 50%; /* Shorthand for grow, shrink, and basis */
}
```

## Grid

- [Tailwind CSS](https://tailwindcss.com/docs/grid-template-columns)
- [W3Schools](https://www.w3schools.com/css/css_grid.asp)
- Grid is a two-dimensional layout model that allows for the placement of elements in rows and columns.
- The CSS Grid Layout should be used for two-dimensional layout, with rows AND columns.

### Grid Parent Properties

```css
section {
  display: grid;
  gap: 5px; /* Space between rows and columns */
  grid-template-rows: 150px 150px 150px; /* Defines 3 rows */
  grid-template-rows: repeat(2, 1fr) 150px; /* Shorthand to repeat rows */

  grid-template-columns: repeat(6, 1fr); /* Defines 6 equal columns */
  grid-template-columns: repeat(3, 1fr) auto 680px; /* 5 columns with specific widths */

  /* Shorthand */
  grid: repeat(3, 1fr) / repeat(6, 1fr); /* Rows / Columns */
}
```

### Grid Child Properties

```css
.box-1 {
  grid-row-start: 1;
  grid-row-end: 3;
  grid-column-start: 2;
  grid-column-end: 4;

  /* Shorthand */
  grid-row: 1 / 3; /* Start at row 1 and span 2 rows */
  grid-column: 2 / span 3; /* Start at column 2 and span 3 columns */

  /* Define both row and column */
  grid-area: row-start / column-start / row-end / column-end;
  grid-area: 1 / 2 / 3 / 4;

  /* Controls the alignment of individual grid items */
  justify-self: center; /* Aligns items horizontally (X-axis) */
  align-self: center; /* Aligns items vertically (Y-axis) */
}
```
